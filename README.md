# C# .NET Framework 4.8 Projects

This repository contains five C# projects covering programming concepts, object-oriented design, console applications, reusable frameworks, and Windows Forms development.

## Projects

### Ex01 - Basic Utilities
A set of introductory console exercises focused on number processing, string analysis, loops, and formatted output.

- **Ex01_01 - Binary Analyzer:** Reads binary values and performs basic calculations and analysis.
- **Ex01_02 - Number Tree:** Prints a fixed numeric pattern/tree.
- **Ex01_03 - Dynamic Number Tree:** Prints a user-defined numeric tree.
- **Ex01_04 - String Analyzer:** Checks string properties such as palindrome and ordering.
- **Ex01_05 - Digit Statistics:** Computes simple statistics for the digits of a number.

### Ex02 - Connect Four Console
A console-based Connect Four game with player turns, board updates, and win/draw logic.

### Ex03 - Garage Manager
A console application for managing vehicles in a garage using object-oriented design and class hierarchies.

### Ex04 - Console Menu Framework
A reusable menu system for console applications, implemented with both interfaces and event-based design.

### Ex05 - Connect Four WinForms
A Windows Forms implementation of Connect Four with a graphical user interface.
